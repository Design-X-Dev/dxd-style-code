# DXDropdownDivider

**Категория:** Molecule  
**Импорт:** `import { DXDropdownDivider } from 'dxd-style-code'`

## Назначение

Разделитель в выпадающем меню для визуального разделения групп элементов.

## Props

Нет props. Компонент не принимает props.

## Примеры использования

```vue
<template>
  <DXDropdown>
    <DXDropdownItem>Пункт 1</DXDropdownItem>
    <DXDropdownItem>Пункт 2</DXDropdownItem>
    <DXDropdownDivider />
    <DXDropdownItem>Пункт 3</DXDropdownItem>
    <DXDropdownItem>Пункт 4</DXDropdownItem>
  </DXDropdown>
</template>
```

## Используется в

- `DXDropdown` - выпадающие меню
- Группировка элементов меню
