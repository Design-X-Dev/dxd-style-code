# Статус документации компонентов

## Компоненты с полной документацией

Эти компоненты имеют полную документацию в формате `parameters.docs.description.component` с разделами:

- Назначение
- Архитектура (Использует / Используется в)
- Внутренняя логика
- Особенности

### Atoms

- ✅ **DX** - универсальный polymorphic компонент
- ✅ **DXFlex** - Flexbox контейнер
- ✅ **DXLink** - компонент ссылки
- ✅ **DXNav** - универсальная обертка для навигации

### Molecules

- ✅ **DXButtonGroup** - группа связанных кнопок
- ✅ **DXBreadcrumb** - хлебные крошки
- ✅ **DXPagination** - пагинация
- ✅ **DXCopyField** - поле для копирования значения
- ✅ **DXTimePicker** - выбор времени
- ✅ **DXStatCard** - карточка статистики

### Organisms

- ✅ **DXReportGenerator** - генератор отчетов
- ✅ **DXSettingsPanel** - панель настроек

## Приоритеты для дополнения документации

### Высокий приоритет (часто используемые компоненты)

1. **DXButton** - базовая кнопка, используется везде
2. **DXInput** - базовое поле ввода
3. **DXInputGroup** - группа полей ввода
4. **DXCard** - карточка, используется для контейнеров
5. **DXModal** - модальное окно
6. **DXTable** - таблица данных
7. **DXTabs** - вкладки
8. **DXMenu** - меню навигации
9. **DXAlert** - уведомления
10. **DXBadge** - бейджи статусов

### Средний приоритет

- DXSelect, DXTextarea, DXCheckbox, DXRadio
- DXIcon, DXAvatar, DXImage
- DXBox, DXStack, DXGrid
- DXSidebar, DXHeaderBar, DXAppLayout

### Низкий приоритет

- Специализированные компоненты (DXChartContainer, DXMediaGallery и т.д.)
- Utility компоненты
- Реже используемые компоненты форм
